# Visualizing Calculus

The goal of this project is to make calculus more intuitive by connecting equations to geometric and graphical interpretations — helping concepts like limits, derivatives, integrals, and series feel less abstract and more visual.

## Open the Notebook
An interactive Jupyter notebook for exploring calculus visually through graphs, animations, and computational examples.

[VisualizingCalculus.ipynb](./VisualizingCalculus.ipynb)

Built with Python, Jupyter, NumPy, and Matplotlib.
